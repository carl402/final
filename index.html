Sistema de Decisiones Empresariales

Monte Carlo Decision Engine

Características

  • Simulaciones Monte Carlo con 10,000+ iteraciones
  • Base de datos PostgreSQL (Neon)
  • Dashboard interactivo con Plotly
  • Análisis de riesgo VaR y CVaR
  • Comparación de escenarios empresariales

Métricas Calculadas

  • NPV (Valor Presente Neto)
  • ROI (Retorno de Inversión)
  • Probabilidad de Éxito
  • Break-even Analysis
  • Value at Risk (VaR 95%)

Tecnologías

Python • NumPy • Pandas • Plotly • Dash • PostgreSQL • Neon

Instrucciones de Uso

  1. Clona el repositorio desde GitHub
  2. Instala dependencias: pip install -r requirements.txt
  3. Configura tu URL de Neon en .env
  4. Ejecuta: python main.py
  5. Abre http://localhost:8050
